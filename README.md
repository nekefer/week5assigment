# E-Commerce Sales Prediction with Neural Networks

This project implements a neural-network regression workflow for predicting e-commerce sales using the UCI Online Retail dataset. It was prepared for the CST 600 neural networks assignment scenario for Zenith Electronics.

## Files

- `Online Retail.xlsx` - source dataset
- `ecommerce_sales_mlp.ipynb` - complete Jupyter Notebook with preprocessing, EDA, MLP baseline, hyperparameter tuning, evaluation, and visualizations
- `requirements.txt` - Python dependencies
- `best_mlp_sales_model.pkl` - saved model generated after running the notebook
- `model_results.csv` - model evaluation metrics generated by the notebook
- `sales_predictions.csv` - actual versus predicted sales values generated by the notebook

## How to Run

```bash
python -m venv .venv
source .venv/Scripts/activate
pip install -r requirements.txt
jupyter notebook ecommerce_sales_mlp.ipynb
```


## Project Summary

The notebook predicts `TotalSales`, calculated as `Quantity * UnitPrice`, after cleaning cancelled invoices, missing values, and invalid transaction rows. Numeric features are standardized, categorical country values are one-hot encoded, and an `MLPRegressor` is trained using both a baseline architecture and tuned hyperparameters.

The final notebook reports Mean Squared Error, Root Mean Squared Error, Mean Absolute Error, and R-squared, then visualizes predicted versus actual sales.
